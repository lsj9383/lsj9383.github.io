1.curl

curl用以模拟http请求，可以用来测试和制作爬虫。

options在下面列出常用的选项:

  • -d 设置请求body，并将请求设置为POST请求
  • -H 设置请求头，每个请求头都要加一个-H。
  • -b 设置Cookie内容, 如:curl -b “a=1;b=2;c=3;”
  • -s 静默模式, 不显示错误信息和请求进度条，这在脚本中常用。
  • -S 不显示请求进度条，但是显示错误信息。
  • -i 输出http响应行(报文协议+响应状态)和响应头
  • -v 输出http请求行(method+url+报文协议)和请求头 + 响应行和响应头
