vim笔记

Author: Arthur Lu Date: August 1, 2019  00:17:18

一、基础

四种模式:

  • Normal
  • Insert
  • Command
  • Visual

二、Nomral模式

Normal模式主要是用于阅读文本。
光标移动:

  • hjkl
  • w/W, 移动到下一个word/WORD开头(word是指非字母作为word的分隔符, WORD代表以空格符作为word的分隔符)
  • b/B, 移动到删一个word/WORD开头 woijojoiw
  • e/E, 移动到下一个word/WORD结尾
  • $, 快速移动到行尾
  • 0, 快速移动到行首
  • ctrl + o, 移动到光标上一个位置
  • ctrl + u, 向上翻页
  • ctrl + f, 向下翻页
  • gg, 移动到页首
  • G, 移动到页尾

搜索:

  • 行内搜索:f{char}, 光标快速跳到匹配的字符。;移动到该行下一个匹配的字符 ,移动到该行上一个匹配的字符。
  • 行内搜索:F{char}, 光标快速跳到上一个匹配的字符。;移动到该行下一个匹配的字符 ,移动到该行上一个匹配的字符。

三、Insert模式

从Normal切换到Insert模式的几种方式:

  • i
  • a
  • o
  • I
  • A
  • O
  • gi

常用快捷键, 非常方便:

  • ctrl + h 删除上一个字符, 类似于Escape
  • ctrl + w 删除上一个单词(删除直到)
  • ctrl + u 删除一行
  • gi 光标回到上次编辑位置, 并进入编辑模式

上述快捷键同样适用于终端操作.这里补充下终端的操作(不适用于vim):

  • ctrl + a 光标快速移动到开头
  • ctrl + e 光标快速移动到结尾
